## About

Tools to make easy and simple GUI for SDL2 in C++.

## Used libraries

| Name | Used for | Included in this repository |
| :------: | :----------: | :---: |
| [SDL2](https://www.libsdl.org/) | GUI | ✅ |
| [stb_image](https://github.com/nothings/stb) | Images | ✅ |
| [stb_truerype](https://github.com/nothings/stb) | Fonts | ✅ |

## Compilation

This wrapper library will compile for 64-bit Linux and Windows out of the box.

### Compiled & Tested on

| OS | Compiler |
| :------: | :----------: |
| Ubuntu | g++ 15.2.0 |
| Windows 10 | MSVC 19.50.35719.0 |

### CMake

   ```bash
   git clone https://github.com/Riyufuchi/ComponentSDL.git
   cd ComponentSDL
   # Run CMake
   cmake -B build -S .
   cmake --build build --config Release
   ```
